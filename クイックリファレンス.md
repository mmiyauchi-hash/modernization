## 🚀 改修作業の基本フロー（5分で理解）

```
1. cd "/Users/minorumiyauchi/Desktop/AI駆動エンジン商談/IGPF/devops-modernization-agent-vite"
2. npm run dev          # 開発サーバー起動（別ターミナル）
3. コードを編集         # Cursorで編集

7. git push
8. Vercelでデプロイ確認 # 数分で自動デプロイ完了
```

## 📝 よく使うコマンド

```bash

```

```bash

git add .

# コミット
git commit -m "修正内容"

# プッシュ
git push
```

3. **デプロイ失敗** → Vercelの「Build Logs」で確認
4. **変更が反映されない** → ブラウザのキャッシュをクリア（`Cmd + Shift + R`）
